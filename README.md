# pypong
Pong in Python

## Install requirements
* `pip install requirements.txt`

## To run
* run `source ./venv/bin/activate`

## To play
* a/d or <-/-> to move left/right
* first to 10 wins

## Opponent Logic
* produces a dx based on the difference between the ball and the platform, capped out at a max speed
